# 🧵 Aai Tailor Management System

A Django-based Tailor Management Web Application.

## 🚀 Features

- Add New Customer
- Old Customers Folder View
- Search Customer by Name
- Customer Detail Page
- Beautiful Pink UI Theme

## 🛠 Tech Stack

- Python
- Django
- HTML & CSS
- SQLite Database
- Git & GitHub

## 📂 Project Structure

Home → Add Customer → Old Customers → Customer Detail

## 📌 New Feature Added

- Order Status System (Pending, In Stitching, Ready, Delivered)
- Mobile Responsive Layout
